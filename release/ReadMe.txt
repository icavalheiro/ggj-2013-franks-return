Frank`s Return

Plataform: Android 4.0

Desenvolvidores: Desmond Quevedo, Ivan Cavalheiro, 
		 Christoffer Oliveira, Laurent Le Avner Quevedo, 
		 Bruno Raupp Kieling e Juan Diego Francke Cardoso

Breve Resumo
Em busca de um coração Frank precisa de sua ajuda para recuperá-lo, 
leve o coração até o corpo moribundo de Frank!!! Frank's Return é 
um puzzle desenvolvido para mobile que utiliza o acelerômetro e touch 
screen, seu objetivo é colocar o coração de volta ao peito de nosso 
personagem, cuidando para não explodir e não parar.

Jogabilidade:

Touch screen para acesso aos menus e movimentação de plataformas e
utilização de acelerometro para a movimentação do coração.

Objetivo:
Lervar o coração ao peito de Frank.

Instalação:

Copiar o arquivo .APK para dentro de uma pasta do seu aparelho com sistema
operacional Android executar o aplicativo.
